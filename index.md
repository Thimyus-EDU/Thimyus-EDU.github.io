## Artifact Overview

The original artifact was a task management application created during earlier coursework. It included basic task management functionality and demonstrated object-oriented programming, validation, and service-layer logic. For CS 499, I expanded the artifact across three major areas of computer science.

The first enhancement focused on software design and engineering. I rewrote and expanded the project into a modular C++ command-line application with separate responsibilities for command parsing, task logic, data modeling, and persistence.

The second enhancement focused on algorithms and data structures. I added searching, filtering, sorting by priority, sorting by due date, and highest-priority task retrieval to make the application more useful and better organized.

The third enhancement focused on databases. I replaced JSON-based persistence with SQLite database storage, added SQL CRUD operations, and used prepared statements to support safer database interaction.
